Welcome to SB MACRO


This program can be used to count the data present in one or multiple projects in the NWCSC or SWCSC directories on Science Base.

Type or paste one or more Science Base Project urls (separated by commas) onto the line below and press "Add" to add them to a running list of projects to be analyzed. Double click an item to delete it. When finished, click "Submit".

You may also upload a CSV of cells that each contain a Science Base Project url.

Any incorrect or unrecognized urls will be ignored.

Do you want us to perform a Hard Search for projects we don't recognize? This option should be turned on if you want information of projects created or modified in the last day (we create and cache ScienceBase data every night).
